# 🚀 Modbus TCP Performance Benchmarking on ifm CR413S Controller

This Project was developed and tested using CODESYS V3.5 SP18 Patch 0.

## 📌 Project Overview
This project focuses on benchmarking and analyzing the **Modbus TCP communication performance** of the **ifm CR413S Controller** using **CODESYS**.  

The goal is to evaluate:
- Communication stability
- Maximum supported Modbus connections
- Throughput performance
- PLC cycle timing
- CPU load under heavy communication traffic


This project demonstrates industrial automation, networking, and real-time communication concepts used in smart manufacturing systems.

---

## 🎯 Objectives

- Configure Modbus TCP Master and Slave communication
- Benchmark communication performance under varying loads
- Analyze throughput and cycle times
- Monitor CPU usage and communication errors

- **ifm CR413S Controller**
